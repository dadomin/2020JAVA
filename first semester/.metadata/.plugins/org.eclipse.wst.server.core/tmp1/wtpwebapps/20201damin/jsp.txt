[JSP]
Java Server Page의 약자
HTML 태그에 자바로 프로그래밍하여 브라우저에 보여주는 특별한 페이지
HTML 파일에는 자바 코드를 작성할 수 없기 때문에 자바 코드를 넣어야 한다면 JSP 페이지로

[JSP 특징]
동적 웹 어플리케이션 구현 가능
확장자는 jsp
클라이언트의 요청에 동적으로 동작을 하며, 응답은 html로 응답
jsp 파일은 서블릿으로 변환되어 실행(JSP 컨테이너 담당)
MVC 패턴에서 view 로 이용(화면 구현이 필요함)
[JSP 기본 태그]
[1] 지시자(directive)
	- 용도 : JSP 페이지의 속성을 지정함
	- 형식 : <%@ %>
	- 종류 : page, include, taglib
[2] 스크립트릿(scriptlet)
	- 용도 : 자바 코드를 기술함
	- 형식 : <% %>
[3] 선언식(declaration)
	- 용도 : 전역변수와 메서드를 선언함
	- 형식 : <%! %>
[4] 표현식(expression)
	- 용도 : 웹브바루저에 문자열 형태로 출력함
	- 형식 : <%= %>
[5] 주석(comment)
	- 용도 : JSP 페이지에 설명을 넣음
	- 형식 : <%-- --%>
[스크립트릿이란?]
JSP 페이지가 서블릿으로 변환되고 요청될 때 _jspService 메소드 안에 서언이 되는 요소.
스크립트릿은 선언무노가 달리 선언된 변수는 지역 변수로 선언이 되고 메소드 선언(선언태그 <%! %>에서 가능)은 할 수 없음.
만약 선언을 하게 되면 메소드 안에 메소드를 선언한 것이기 때문에 만들 수가 없음.
[스크립트릿 문법]
<% 지역변수 선언, if, for, while, switch 문%>
